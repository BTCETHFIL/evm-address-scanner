# EVM地址扫描器

## 项目简介

EVM地址扫描器是一个用于生成EVM兼容地址并与地址库进行比较的工具。该项目是使用AI工具实现的第一个程序，旨在展示AI在软件开发中的应用能力。

## 功能特性

- **地址生成**：根据BIP39协议生成EVM兼容地址和对应的助记词
- **地址比较**：与给定的地址库进行比较，发现匹配的地址
- **结果记录**：记录匹配的地址和对应的助记词
- **性能优化**：多线程并行处理，内存映射文件处理大地址库
- **资源控制**：限制CPU和内存使用，确保系统稳定
- **文件管理**：自动分割大文件，定期合并和清理
- **用户界面**：简单直观的图形界面，易于操作

## 系统要求

- **操作系统**：Windows 11 64位
- **处理器**：AMD Ryzen 7 7735HS或同等性能
- **内存**：至少16GB RAM
- **存储空间**：至少10GB可用空间
- **Python版本**：Python 3.10或更高版本

## 安装步骤

1. **克隆项目**：
   ```bash
   git clone https://github.com/your-username/evm-address-scanner.git
   cd evm-address-scanner
   ```

2. **安装依赖**：
   ```bash
   pip install -r requirements.txt
   ```

3. **运行程序**：
   ```bash
   python evm_address_scanner.py
   ```
   或双击运行 `run.bat` 脚本

## 使用方法

1. **导入地址库**：点击"导入数据库"按钮，选择一个或多个包含EVM地址的文件（支持.txt和.csv格式）

2. **开始扫描**：点击"开始"按钮，程序将开始生成EVM地址并与地址库进行比较

3. **暂停/继续**：点击"暂停"按钮可以暂停扫描，再次点击"继续"按钮恢复扫描

4. **停止扫描**：点击"停止"按钮停止扫描过程

5. **查看结果**：点击"查看结果"按钮查看匹配到的地址和对应的助记词

6. **合并和清理**：点击"合并和清理"按钮合并小文件并清理重复地址

## 技术实现

- **地址生成**：使用`eth-account`库和`mnemonic`库生成EVM地址和助记词
- **多线程**：使用Python的`threading`模块实现并行处理
- **内存映射**：使用`mmap`模块处理大地址库文件
- **用户界面**：使用`tkinter`库创建图形界面
- **资源监控**：使用`psutil`库监控系统资源使用情况

## 性能优化

- **CPU使用**：限制为8个线程（约50%的16线程CPU）
- **内存使用**：限制为16GB（约50%的32GB内存）
- **批量处理**：使用5000个地址的批量处理，减少I/O操作
- **索引结构**：为生成的地址创建索引，提高查询效率
- **文件管理**：自动分割大文件，定期合并和清理

## 注意事项

- 生成地址的速度取决于硬件性能
- 地址库文件越大，加载时间越长
- 程序会自动保存生成的地址和助记词，确保数据不会丢失
- 匹配结果会保存在`results.txt`文件中
- 定期使用"合并和清理"功能可以保持文件系统整洁
